# Game of thrones API

Game of thrones characters API.

## Getting Started

### Install

Clone the repository and execute
```
make up
```
After that, execute
```
make install
```

Your app will be running on 127.0.0.1:80. If you want to run it with a host, just add it to your host file
```
127.0.0.1       local.got.test
```
